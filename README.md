# Node + Fastify
This is a node application based on fastify and ejs, with modern node resources for typescript, tests and file watcher (for development server).

## Scripts

- Run tests:
```shell
npm run test
```

- Run tests and generate code coverage:
```shell
npm run test:cov
```

- Start:
```shell
npm run start
```

- Start (development mode):
```shell
npm run start:dev
```
